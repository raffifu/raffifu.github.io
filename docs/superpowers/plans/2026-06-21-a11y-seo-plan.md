### Task 5: Single Page Template (`layouts/_default/single.html`)

**Files:**
- Modify: `layouts/_default/single.html`

- [ ] **Add JSON-LD BlogPosting structured data**

Right after `{{ define "main" }}` (line 1), add:
```html
    {{ if eq .Type "post" }}
    <script type="application/ld+json">
    {
        "@context": "https://schema.org",
        "@type": "BlogPosting",
        "headline": "{{ .Title }}",
        "description": "{{ if .Description }}{{ .Description | plainify }}{{ else }}{{ .Summary | plainify }}{{ end }}",
        "author": {
            "@type": "Person",
            "name": "{{ if .Params.author }}{{ .Params.author }}{{ else }}{{ .Site.Params.author }}{{ end }}"
        },
        "datePublished": "{{ .PublishDate.Format "2006-01-02T15:04:05Z07:00" }}",
        "dateModified": "{{ .Lastmod.Format "2006-01-02T15:04:05Z07:00" }}",
        "mainEntityOfPage": {
            "@type": "WebPage",
            "@id": "{{ .Permalink }}"
        },
        {{ if .Params.tags }}"keywords": [{{ range $i, $tag := .Params.tags }}{{ if $i }}, {{ end }}"{{ $tag }}"{{ end }}],{{ end }}
        {{ if .Params.thumbnail }}"image": "{{ .Params.thumbnail | absURL }}",{{ end }}
        "url": "{{ .Permalink }}"
    }
    </script>
    {{ end }}
```

- [ ] **Improve thumbnail alt text**

Change:
```html
        <img src="{{ .Params.thumbnail }}" alt="Post thumbnail" class="img-fluid rounded shadow-sm" style="width: 100%; object-fit: cover;" />
```
To:
```html
        <img src="{{ .Params.thumbnail }}" alt="Thumbnail for: {{ .Title }}" class="img-fluid rounded shadow-sm" style="width: 100%; object-fit: cover;" />
```

- [ ] **Add `role="button"`, `tabindex="0"`, and `aria-label` to Copy Link span**

Change:
```html
            <span id="copyLinkBtn" style="cursor: pointer;" class="text-decoration-none">
                <i class="fas fa-link me-2"></i><span id="copyLinkText">Copy Link</span>
            </span>
```
To:
```html
            <span id="copyLinkBtn" style="cursor: pointer;" class="text-decoration-none" role="button" tabindex="0" aria-label="Copy link">
                <i class="fas fa-link me-2"></i><span id="copyLinkText">Copy Link</span>
            </span>
```

- [ ] **Add live region for screen reader announcement**

After the copy link `</div>` (before the script), add:
```html
    <div id="copyLinkAnnounce" class="visually-hidden" aria-live="polite"></div>
```

- [ ] **Update copy link script to announce to screen readers**

Update the script block:
```html
    <script>
        document.getElementById('copyLinkBtn').addEventListener('click', function() {
            navigator.clipboard.writeText(window.location.href).then(function() {
                const textSpan = document.getElementById('copyLinkText');
                const originalText = textSpan.innerText;
                textSpan.innerText = 'Copied!';
                document.getElementById('copyLinkAnnounce').textContent = 'Link copied';
                setTimeout(() => {
                    textSpan.innerText = originalText;
                    document.getElementById('copyLinkAnnounce').textContent = '';
                }, 2000);
            });
        });
    </script>
```
